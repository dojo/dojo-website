dojox/app/utils/nls (version 1.9)

nsl is called to create to load the nls all for the app, or for a view.

Usage:

nls(config,parent);
Parameter Type Description
config Object

The section of the config for this view or for the app.
parent Object

The parent of this view or the app itself, so that models from the parent will be available to the view.
Returns:boolean
