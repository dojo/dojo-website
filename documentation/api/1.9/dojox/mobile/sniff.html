dojox/mobile/sniff (version 1.9)

Deprecated: use dojo/sniff instead. On iOS, dojox/mobile/sniff sets "iphone" to the same value as "ios" for compatibility with earlier versions, but this should be considered deprecated. In future versions, "iphone" will be set only when running on an iPhone (not iPad on iPod).
