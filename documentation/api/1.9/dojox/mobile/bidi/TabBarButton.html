dojox/mobile/bidi/TabBarButton (version 1.9)

Support for control over text direction for mobile TabBarButton widget, using Unicode Control Characters to control text direction.

This class should not be used directly. Mobile TabBarButton widget loads this module when user sets "has: {'dojo-bidi': true }" in data-dojo-config.
