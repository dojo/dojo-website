dojox/mobile/bidi/TreeView (version 1.9)

Support for control over text direction for mobile TreeView widget, using Unicode Control Characters to control text direction.

Text direction attribute of the Tree is set to ListItem. This class should not be used directly. Mobile TreeView widget loads this module when user sets "has: {'dojo-bidi': true }" in data-dojo-config.
