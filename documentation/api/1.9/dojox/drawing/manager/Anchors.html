dojox/drawing/manager/Anchors (version 1.9)

Usage:

var foo = new Anchors(options);
Defined by dojox/drawing/manager/Anchors
Parameter Type Description
options dojox.__stencilArgs

Method Summary

  • add(item) Creates anchor points on a Stencil, based on the Stencil's points.
  • remove(item) Destroys the anchor points for a Stencil.

Event Summary

  • onAddAnchor ,Event fires when anchor is created
  • onAnchorDown ,Event fired on anchor mousedown
  • onAnchorDrag ,Event fired when anchor is moved
  • onAnchorUp ,Event fired on anchor mouseup
  • onChangeStyle ,if the Stencil changes color while were's selected this moves the anchors to the back.
  • onRenderStencilEvent fires when an anchor calls a Stencil's render method
  • onReset ,Event fires when an anchor's reset method is called
  • onTransformPoint ,Event fired on anchor drag
