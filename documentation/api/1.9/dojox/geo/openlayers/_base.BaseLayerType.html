dojox/geo/openlayers/_base.BaseLayerType (version 1.9)

Defines the base layer types to be used at Map construction time or with the setBaseLayerType function.

This object defines the base layer types to be used at Map construction time or with the setBaseLayerType function.

Property Summary

  • ARCGIS
    • The ESRI ARCGis base layer selector.
  • BING

  • GOOGLE

  • OSM
    • The Open Street Map base layer type selector.
  • VIRTUAL_EARTH
    • The Virtual Earth base layer type selector.
  • WMS
    • The Web Map Server base layer type selector.
  • YAHOO
    • The Yahoo base layer type selector.
