dojox/mvc/getPlainValue (version 1.9)

Create a raw value from a dojo/Stateful object.

Recursively iterates the stateful value given, and convert them to raw ones.

Usage:

getPlainValue(value,options);
Parameter Type Description
value Anything

The stateful value.
options dojox/mvc/getPlainValueOptions

The object that defines how plain value should be created from stateful value.
Returns:any | undefined

The converted value.

Method Summary

  • getPlainArray(a) Returns the stateful version of the given array.
  • getPlainObject(o) Returns the stateful version of the given object.
  • getPlainValue(v) Just returns the given value.
  • getType(v) Returns the type of the given value.
