dojox/form/manager/_DisplayMixin (version 1.9)

Form manager's mixin for controlling show/hide state of controlled elements (defined by dojoAttachPoint attributes).

This mixin provides unified show/hide functionality for controlled elements (indicated by dojoAttachPoint attribute). Essentially it provides a way to change "style.display" parameter of controlled nodes. It should be used together with dojox.form.manager.Mixin.

Method Summary

  • gatherDisplayState(names) Gather display state of all attached elements and return as a dictionary.
  • hide(state) Hide attached nodes according to the supplied state object.
  • show(state,defaultState) Show attached nodes according to the supplied state object.

Methods

gatherDisplayState(names)
Defined by dojox/form/manager/_DisplayMixin

Gather display state of all attached elements and return as a dictionary.

Parameter Type Description
names Object
Optional

If it is an array, it is a list of names to be processed. If it is an object, dictionary keys are names to be processed. If it is omitted, all known attach point nodes are to be processed.
Returns:undefined
hide(state)
Defined by dojox/form/manager/_DisplayMixin

Hide attached nodes according to the supplied state object.

Parameter Type Description
state Object
Optional

Optional. If a name-value dictionary, the value is true to show and false to hide. If an array, all names in the array will be hidden. If omitted, all form elements will be hidden.
Returns:undefined
show(state,defaultState)
Defined by dojox/form/manager/_DisplayMixin

Show attached nodes according to the supplied state object.

Parameter Type Description
state Object
Optional

Optional. If a name-value dictionary, the value is true to show and false to hide. If an array, all names in the array will be set to defaultState. If omitted, all form elements will be set to defaultState.
defaultState Boolean
Optional

The default state (true, if omitted).
Returns:function

Show attached nodes according to the supplied state object.
