dojox/mobile/TimePicker (version 1.10)

A wrapper widget around SpinWheelTimePicker or ValuePickerTimePicker. Returns ValuePickerTimePicker when the current theme is "android" or "holodark". Returns SpinWheelTimePicker otherwise.

Usage:

TimePicker();

See thedojox/mobile/TimePicker reference documentation for more information.
