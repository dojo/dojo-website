dojox/gfx/_base.Moveable (version 1.8)

Usage:

var foo = new _base.Moveable(shape,params);
Defined by dojox/gfx/Moveable
Parameter Type Description
shape dojox/gfx.Shape

a shape object to be moved.
params Object

an optional configuration object.

Method Summary

  • destroy() stops watching for possible move, deletes all references, so the object can be garbage-collected

Event Summary

  • onFirstMove ,called during the very first move notification, can be used to initialize coordinates, can be overwritten.
  • onMouseDown ,event processor for onmousedown, creates a Mover for the shape
  • onMouseMove ,event processor for onmousemove, used only for delayed drags
  • onMouseUp ,event processor for onmouseup, used only for delayed delayed drags
  • onMove , ,called during every move notification, should actually move the node, can be overwritten.
  • onMoved , ,called after every incremental move, can be overwritten.
  • onMoveStart ,called before every move operation
  • onMoveStop ,called after every move operation
  • onMoving , ,called before every incremental move, can be overwritten.
