dojox/gfx/_base.defaultImage (version 1.8)

Defines the default Image prototype.

Property Summary

  • height
    • The height of the image, default value 0.
  • src
    • The src url of the image, defaults to empty string.
  • type
    • Specifies this object is an image, value &#39;image&#39;.
  • width
    • The width of the image, default value 0.
  • x
    • The X coordinate of the image&#39;s position, default value 0.
  • y
    • The Y coordinate of the image&#39;s position, default value 0.
