dojox/charting/widget/BidiSupport (version 1.8)

Add support to dojox/charting/widget for bidi scripts.

Bidi stands for support for languages with a bidirectional script. There's a special need for displaying BIDI text in rtl direction in ltr GUI, sometimes needed auto support. dojox.charting does not support control over base text direction provided in Dojo.
